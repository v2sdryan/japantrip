# Gemini Banana Poster Prompts

共同要求：直式 4:5 或 3:4 大 poster，一日一張圖。整張圖要像日本親子旅遊手繪資訊圖，不要分開多張小圖。所有行程文字、時間、每一站去下一站的車程時間、公里數、美食、住宿、親子提醒都要放入同一張圖內。繁體中文、港式用字，標題清楚，字要大、黑、易讀，不要亂碼，不要簡體字。風格：柔和水彩、可愛家庭自駕、北海道夏天、薰衣草紫色、清新藍綠色、少量橙黃色點綴。版面要有虛線自駕路線、汽車 icon、景點插畫、食物插畫、住宿 icon、底部三個框：「今日重點」「美食建議」「自駕提醒」。
